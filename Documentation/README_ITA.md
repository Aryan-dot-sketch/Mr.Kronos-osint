# 🔮 Mr.Kronos v3.0 — Il Framework OSINT Completo

**Mr.Kronos è un avanzato strumento di raccolta informazioni (OSINT) con un'interfaccia completamente ridisegnata. Lo scopo principale è ottenere intelligence su domini, username, email e numeri di telefono usando fonti pubbliche. Sfrutta Google/Yandex Dorks per ricerche mirate, proxy per richieste anonime e un'API WHOIS per l'intelligence sui domini.**

# ✅ INSTALLAZIONE LINUX/MAC:
```bash
git clone https://github.com/aryanhubhaimai/Mr.Kronos-OSINT
cd Mr.Kronos-OSINT
sudo apt-get update
sudo chmod +x install.sh
sudo bash install.sh
```
<br>

# ✅ INSTALLAZIONE WINDOWS:
```cmd
git clone https://github.com/aryanhubhaimai/Mr.Kronos-OSINT
cd Mr.Kronos-OSINT
Install.cmd
```
<br>

# ✅ INSTALLAZIONE TERMUX:
```bash
pkg install proot
git clone https://github.com/aryanhubhaimai/Mr.Kronos-OSINT
cd Mr.Kronos-OSINT
proot -0 chmod +x install_Termux.sh
./install_Termux.sh
```
